API-сервис для бронирования и размещения жилья

Цель проекта
Разработать REST API сервис для бронирования и размещения жилья, позволяющий пользователям находить, бронировать и размещать жилье, а также управлять бронированиями.



Основные требования
Язык программирования: Python
Фреймворк: Django Rest Framework
База данных: PostgreSQL (рекомендуется использовать для работы с геоданными). Но тут можно sqlite пока
Формат данных: JSON


Функциональные требования

Основные функции
Пользователи:

Разделение пользователей на:

1.Гостей (могут бронировать жилье).
2.Владельцев жилья (могут размещать жилье).
3.Профиль пользователя с возможностью редактирования.

Размещение жилья:
Владельцы могут добавлять объявления о жилье:

1.Название, описание, фотографии, адрес, цена за ночь.
2.Опциональные поля: удобства, правила проживания, тип жилья.
3.Возможность редактирования и удаления объявлений.

Поиск жилья:
Фильтрация по:

1.Дате заезда и выезда.
2.Локации (по географическим координатам).
3.Цене.
4.Количеству гостей.

Бронирование жилья:

1.Гости могут бронировать жилье.
2.Проверка доступности жилья на выбранные даты.
3.Возможность отмены бронирования.

Отзывы:

1.Гости могут оставлять отзывы и оценки жилья.
2.Средний рейтинг отображается для каждого жилья.

Админка:

1.Управление пользователями, объявлениями, бронированиями и отзывами.

Структура приложения
Модели
User (AbstractUser):

Поля: email, username, first_name, last_name, is_owner (булево, является ли пользователь владельцем жилья).


Property (жилье):

Поля: title, description, price_per_night, address, latitude, longitude, owner (ссылка на пользователя), photos (прикрепленные изображения), amenities (JSONField для списка удобств), rules.


Booking (бронирование):

Поля: property (ссылка на жилье), guest (ссылка на пользователя), check_in, check_out, total_price, status (например, "подтверждено", "отменено").


Review (отзывы):

Поля: property (ссылка на жилье), user (ссылка на гостя), rating (оценка от 1 до 5), comment.


Роутеры и ViewSet
Users: UserViewSet с регистрацией, входом и просмотром профиля.
Properties: PropertyViewSet для создания, просмотра, обновления и удаления объявлений.
Bookings: BookingViewSet для бронирования жилья, отмены и просмотра бронирований.
Reviews: ReviewViewSet для добавления, редактирования и просмотра отзывов.

Роуты
api/v1/users/: управление пользователями.
api/v1/properties/: работа с жильем.
api/v1/bookings/: управление бронированиями.
api/v1/reviews/: работа с отзывами.
